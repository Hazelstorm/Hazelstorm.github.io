    Sisyphus' Speedway


    Godot, 8 day game jam. We placed 12th out of 276!


    I worked with a team of two programmers and two artists. The game is a bullet-hell racing roguelike.


    I Programmed:

    • Obstacles and upgrades.
    • Upgrade screen and intro screen.
    • Gameplay loop, with a state machine.
    • Bullet spawning system, bullet collision system.
    • Obstacle targeting system that allows previewed obstacles to target the mouse cursor.
    • Shaders for scrolling background and screen-wipe transition.
    • Integrated art files and sprites from artists.

    I Designed:

    • Obstacles and upgrades.
    • Roguelike upgrade system with a pool that changes over time as upgrades are unlocked.
    • Upgrade screen UI.

    I also worked with Patashu to compose the game's soundtrack using Jummbox and FamiTracker.

    Grid Wizard


    Godot, 4 day game jam.


    I worked with several UI designers and arists. The game is a deckbuilder roguelike where you have to arrange cards on a grid in order to use them.


    I Programmed:

    • Block rotation and placement.
    • Card movement (dealing, shuffling, hand position, etc.).
    • Combat gameflow to handle player's turn, enemies' turn, win/loss detection.
    • Upgrade screen.
    • Card effects and enemy actions.
    • Status effect system.
    • Save system for saving and loading in-progress runs.
    • Integrated art files and sprites from artists.

    I Designed:

    • Card effects.
    • Enemy designs, movesets, AI.

    I also composed the game's soundtrack using Jummbox.
